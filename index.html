DESCARGAS ATMOSFÉRICAS

ESTO ES UN PROYECTO ENFOCADO EN LA PREDICCIÓN DE DESCARGAS ATMOSFÉRICAS EN ASUNCIÓN

GOOGLE DRIVE MÁS INFORMACIÓN

QUÉ HICIMOS

EL PROYECTO SE DIVIDE EN 3 BLOQUES.

CÓDIGO FUENTE

El proyecto se desarrollo utilizando Jupyter Notebook, con el entorno Google Colab y Visual Studio Code.

DATOS

Para realizar las predicciones se tomaron datos proveidos por la Dirección de Meterología e Hidrología.

DOCUMENTACIÓN

Procesos realizados a la hora de filtrar y predecir descargas atmosféricas en la ciudad de Asunción.

SECCIÓN DE IMÁGENES

Imágenes extraídas de los Jupyter Notebooks.
